# Open Design H5 Final

This is a static H5 page exported from Open Design.

## Local Preview

Run the following command in this directory:

```bash
python -m http.server 5178
```

Then open:

```text
http://127.0.0.1:5178/
```

## Deployment

This project can be deployed as a static site. Upload `index.html` and the `assets/` folder to any static hosting service.
